# SEO & Structured Data

## Meta Tags

Page-level SEO meta tags are managed through Vue Router's `meta` property. Each route defines:

- `title` — Sets `<title>` and `og:title`
- `description` — Sets `<meta name="description">` and `og:description`

The `router.afterEach` hook in `src/main.js` applies these on every navigation, also updating:
- Canonical URL (`<link rel="canonical">`)
- Open Graph URL (`og:url`)
- GA4 virtual pageview event

## Open Graph

Static OG tags in `index.html`:
- `og:type`: website
- `og:image`: `/og-image.png` (1200x630, generated by `scripts/generate-brand-assets.py`)

Dynamic OG tags updated per route:
- `og:title`, `og:description`, `og:url`

## JSON-LD Structured Data

### Global (index.html)

Two schemas are embedded in the app shell:

- **Organization** — company name, URL, logo, areas served (CA, US, BR), contact point
- **WebSite** — with SearchAction for potential sitelinks search box

### Per-Page (via useJsonLd composable)

The `src/composables/useJsonLd.js` composable injects/updates `<script type="application/ld+json">` tags reactively:

- **ProductsPage**: BreadcrumbList + ItemList (all products)
- **ContactPage**: BreadcrumbList

Usage:
```js
import { useJsonLd } from '@/composables/useJsonLd';

useJsonLd({
  '@context': 'https://schema.org',
  '@type': 'BreadcrumbList',
  // ...
});
```

## Static SEO Files

| File | Purpose |
|------|---------|
| `public/robots.txt` | Allows all crawlers, references sitemap |
| `public/sitemap.xml` | Lists all 3 pages with priorities and change frequencies |
| `public/llms.txt` | Plain-text site description for LLM crawlers |

## Google Analytics 4

GA4 is loaded in `index.html` with measurement ID `G-HX3R7GECKL`. Virtual pageviews are fired on each SPA navigation via `router.afterEach`. The contact form also fires a `form_submission` event on success.

## Favicons

Multiple favicon formats are provided for broad browser/device support:
- `favicon.svg` — modern browsers
- `favicon-16x16.png`, `favicon-32x32.png` — legacy browsers
- `apple-touch-icon.png` — iOS home screen (180x180)
